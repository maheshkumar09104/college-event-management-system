# College Event Management System

This project is a web-based event management system that allows students to register for college events.

## Features
- User registration and login
- Event creation and management
- QR code generation for event registration
- Admin dashboard

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Deployment
The project is deployed online using InfinityFree.
